# 🏏 IPL Dashboard App

A React-based IPL Dashboard application that displays all IPL teams and their match details using API integration.

---

## 🚀 Live Features

- View all IPL teams
- Click on a team to see match details
- Latest match information
- Match result statistics
- Responsive design
- Clean UI

---

## 🛠️ Tech Stack

- React 17
- React Router DOM v5
- React Loader Spinner
- Recharts
- CSS3

## 📦 Installation

Clone the repository:


git clone https://github.com/Laharivanaja/IPL-Dashboard-App.git


Go to project directory:


cd IPL-Dashboard-App


Install dependencies:


npm install


Start the development server:


npm start
